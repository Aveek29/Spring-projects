# Spring Project Workspace

A comprehensive collection of Spring Framework learning projects organized by topic.

## Structure

- **01-Spring-Core/** - Spring Core basics, XML/Java config, DI, Autowiring
- **02-Spring-Boot-Basics/** - Getting started with Spring Boot
- **03-Spring-Web/** - Spring Boot Web, JSP, Thymeleaf, MVC without Boot
- **04-Spring-REST/** - REST APIs, JWT Authentication
- **05-Spring-Data/** - Spring JDBC, JPA, Data Rest
- **06-Spring-Security/** - Spring Security, OAuth2
- **07-Spring-AOP/** - Aspect Oriented Programming
- **08-Spring-MVC-Projects/** - Full MVC applications (JobApp, Ecommerce)
- **09-Servlet-Projects/** - Servlet2 (embedded Tomcat example)
- **10-React-UI/** - React frontend UIs (AllPosts, JOBPORTAL)
